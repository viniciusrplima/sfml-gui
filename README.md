# SFML GUI

Simple graphic interface write in c++ using [SFML](https://www.sfml-dev.org/).

### Dependencies

To install all dependencies:

```
sudo apt install make libsfml-dev
```

### Compilation

To build example program:

```
make build
```

### Run

To run example program:

```
make run
```
or
```
./SFMLGuiTest
```
